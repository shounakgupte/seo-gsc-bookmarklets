🎯 Core Intent Filters

1️⃣ Commercial Intent High Value

Finds queries with buying signals: price, deal, review, best, etc.

\b(buy|price|cost|cheap|deal|discount|sale|best|top|review|vs|comparison)\b
🖱️ GSC: 1️⃣ Commercial Intent
javascript:(function(){var r='\\b(buy|price|cost|cheap|deal|discount|sale|best|top|review|vs|comparison)\\b';var u=new URL(window.location.href);u.searchParams.set('query','~'+r);window.location.href=u.toString();})();

🔧 Regex Syntax Reference (Google RE2)

Pattern Meaning Example
\bWord boundary\bbuy\b matches "buy" not "buyer"
^ / $Start / end of string^how = starts with "how"
|OR operatorbuy|purchase = either term
.*Any characters (greedy).*seo.* = contains "seo"
(?i)Case-insensitive flag(?i)^how matches "How", "HOW"
{n,}N or more repetitions(\w+\s){5,} = 5+ words
!~Does NOT match (prefix)Use query=!~PATTERN to exclude
\s / \SWhitespace / non-whitespace\S+ = one or more non-space chars
⚠️ Important Limitations:
  • Google Search Console uses RE2 regex syntax (not full PCRE)
  • Max 4096 characters per filter pattern
  • No backreferences (\1), limited lookahead/lookbehind
  • Always test patterns in GSC UI before saving as bookmarklet
  • Bookmarklets only work on GSC Performance report pages
  • Firefox users: May see security warning for javascript: URLs - click "Allow"
